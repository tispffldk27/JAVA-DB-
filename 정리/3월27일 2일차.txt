반복문 
- 어떤 일을 반복적으로 할 때 사용하는 문법.
- 규칙이 있어야 한다.

for (초기화 ; 조건식 ;증감식) {
	실행문;
}

조건식 : 조건식을 만족(참이라면)하면 실행문을 실행하고
	만족하지 않으면 반복문을 종료
초기화: 조건식에서 사용할 변수를 초기화 하거나, 실행문에서
	사용할 변수를 초기화한다
증감연산: 조건식에서 사용하는 변수를 증가 또는 감소 시키는 작업을 한다.
