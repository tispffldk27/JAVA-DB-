재귀메소드:
메소드 오버로딩
- 한 클래스에서 동일한 메소드명을 가진 메소드가 여러개인 경우
-조건
1.매개변수의 갯수가 다른경우
2.매개변수의 갯수가 같지만 자료형이 다른경우

클래스 = 맴버변수 + 맴버메소드

맴버변수 : 일반적으로 접근제한자를 private으로하고
맴버메소드 : 일반적으로 접근제한자를 public으로 한다
맴버변수의 값을 변경하려면 맴버 메소드를 이용해서 변경한다.

접근제한자
public : 전부(아무나)
private : 본인
default :같은패키지
protected : 본인 +자식(파생)클래스

public > default >protected > private


같은 패키지 안에 똑같은 클래스 이름이 있으면 안된다.


클래스 멤버 VS 객체 멤버
객체 멤버는 객체가 생성된 후에 사용할 수있다.
객체.멤버변수   ,   객체.맴버메소드로 호출한다 .    앞에 객체가 필요한다
객체 멤버는 만들어진 객체마다 가지고 있다

-클래스 멤버
객체의 생성 없이 사용할 수 있다.
클래스.멤버변수,   클래스.멤버메소드로 호출한다.       앞에 클레스가 필요함
클래스 멤버는 하나씩만 만들어진다.
  
클래스 멤버 메소드에는 클래스 멤버 변수는 사용 가능하지만
객체 멤버 변수는 사용할 수 없다.
